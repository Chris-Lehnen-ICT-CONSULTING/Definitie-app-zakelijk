# Requirement REQ-106: Bronconfiguratie Export & Import

## Beschrijving
Exporteer geregistreerde bronconfiguraties naar JSON (download) en importeer configuraties. Minimale implementatie (handmatige herconfiguratie toegestaan) met duidelijke UX en foutafhandeling.

## Acceptatiecriteria (SMART)
- Specifiek: exportbestand met metadata (timestamp, broninfo); import valideert structuur
- Meetbaar: exportbestand < 1 MB; import foutmeldingen expliciet
- Acceptabel: geen secrets in export zonder expliciete toestemming; waarschuwingen tonen
- Relevant: versimpelt beheer en uitrol naar andere omgevingen
