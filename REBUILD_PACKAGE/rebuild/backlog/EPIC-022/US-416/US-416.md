# US-416: Bulk import met opties en voortgang

## Gebruikersverhaal
Als data‑steward wil ik in bulk definities kunnen importeren vanuit een gekozen bron, met importopties en voortgang, zodat ik efficiënter grote datasets kan verwerken.

## Acceptatiecriteria
- Bronselectie + maximum (limiet) invoer; start bulk import
- Opties: auto‑validatie, status, metadata behouden, overwrite policy
- Voortgangsbalk + tellers; samenvattende resultaten en foutenlijst
- Importgeschiedenis (laatste 5) zichtbaar met tijdstip en status
