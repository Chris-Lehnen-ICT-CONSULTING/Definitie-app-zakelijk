## Extracted Knowledge Summary

### 1. FeedbackBuilder Algoritmes
- 5-staps prioritering algoritme
- Violation-to-feedback mapping (11+ mappings)
- Iteratie-bewuste feedback
- Stagnatie detectie

### 2. Status Bepaling
- 3-staps acceptatie criteria
- Weighted scoring algoritme
- Severity multipliers

### 3. Iterative Improvement
- Max 3 iteraties, 0.05 threshold
- Voorbeelden caching optimalisatie
- Best iteration tracking
- Feedback history FIFO

### 4. Critical Thresholds
```python
acceptance = {"overall": 0.8, "critical": 0, "category": 0.75}
iteration = {"max": 3, "threshold": 0.05, "feedback": 5}
enhancement = {"max": 3, "confidence": 0.6, "vagueness": 0.3}
```
