US-440: Handmatige bronconfiguratie (ID/naam/type/verbinding)

Gebruikersverhaal

Als integratie‑specialist wil ik een nieuwe bron handmatig kunnen configureren (id, naam, type, connection string, evt. API key, timeout) zodat ik bronnen zonder upload kan toevoegen.

Acceptatiecriteria

  • Formulier met verplichte velden; validatie en duidelijke foutmeldingen
  • API‑sleutels niet zichtbaar in plain tekst nadat ingevuld (maskeren)
  • Verbindingstest bij aanmaken; registratie in manager na succes
