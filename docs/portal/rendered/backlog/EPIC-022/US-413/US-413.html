US-413: Bronbeheer UI (registreren, testen, verwijderen)

Gebruikersverhaal

Als beheerder wil ik externe bronnen kunnen registreren, de verbinding testen en bronnen kunnen verwijderen zodat ik gecontroleerd bronnen kan activeren/deactiveren.

Acceptatiecriteria

  • Lijst toont alle bronnen met id, naam, type en status (verbonden/niet)
  • Actieknoppen: Test verbinding, Verwijder bron
  • Mock‑bron toevoegen (1 klik)
  • Bestand‑bron toevoegen via upload (CSV/JSON), daarna registeren
  • Fouten worden zichtbaar getoond (geen witte schermen)

Notities

  • Manager persistent in sessiestatus; veilige herinitialisatie
  • Log duidelijke meldingen bij connect/test failures
