US-127: Externe API (REST)

Niet nodig voor single‑user; gepland voor multi‑user. Externe endpoints (read/write) met OAS en rate limiting.

Acceptatiecriteria (functioneel)

  • [ ] OAS (v3) skeleton gepubliceerd met basis endpoints (read/write)
  • [ ] Versiebeheer (v1) + backward compatible evolution rules
  • [ ] Error‑model met codes en trace‑id in responses

Niet‑functioneel

  • [ ] Rate limiting per key/endpoint
  • [ ] Minimal logging (geen PII), correlation id naar logs

Definition of Done

  • [ ] CI publiceert OAS artefact
  • [ ] Contract test (rooktest) in CI
