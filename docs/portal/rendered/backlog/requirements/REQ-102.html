Requirement REQ-102: External Sources Adapter Framework

Beschrijving

Implementeer een uitbreidbaar adapter‑framework (ExternalSourceAdapter) voor externe definitiebronnen met uniforme operaties: connect, test_connection, search_definitions, get_definition, create/update/delete_definition en get_source_info. Lever ten minste een Mock‑adapter en een File‑system adapter.

Rationale

Een generiek framework maakt toevoegen/uitwisselen van bronnen mogelijk zonder UI‑wijzigingen en versnelt adoptie/migratie.

Acceptatiecriteria (SMART)

  • Specifiek: abstracte basis + ten minste 2 concrete adapters (mock, file)
  • Meetbaar: adapters geregistreerd via `ExternalSourceManager` en zichtbaar in UI
  • Acceptabel: type‑veilige config (`ExternalSourceConfig`), duidelijke foutmeldingen
  • Relevant: ontsluit externe bronnen breed voor zoeken/import
  • Tijdgebonden: MVP bij EPIC‑022 oplevering

Extra details

  • Retries/timeout velden in config; logging van connect/search fouten
  • Manager: register/unregister/get/search_all_sources, lazy connect
