Business Context

Analyseert definities op ontbrekende aspecten zoals doel, scope, voorwaarden, proces, en verantwoordelijkheid.

Acceptance Criteria

  • [ ] Check 5 hoofdaspecten (doel, scope, voorwaarden, proces, verantwoordelijk)
  • [ ] Detecteer missing aspects via keyword matching
  • [ ] Trigger enhancement bij ≥2 missing aspects
  • [ ] Genereer specifieke completeness hints
  • [ ] Integreer natuurlijk in definitie

Completeness Aspects

aspects = {
    "doel": ["doel", "bedoeling", "functie", "nut"],
    "scope": ["omvang", "bereik", "toepassingsgebied"],
    "voorwaarden": ["voorwaarde", "vereiste", "criteria"],
    "proces": ["stap", "procedure", "methode", "proces"],
    "verantwoordelijk": ["verantwoordelijk", "bevoegd", "eigenaar"]
}

Detection Logic

def analyze_completeness(definitie: str) -> list[str]:
    missing = []
    text_lower = definitie.lower()
    
    for aspect, keywords in aspects.items():
        if not any(kw in text_lower for kw in keywords):
            missing.append(aspect)
    
    return missing

Enhancement Suggestions

  • Doel + Proces missing: "Het [begrip] heeft een specifiek doel en volgt een bepaalde procedure."
  • Scope missing: "Het toepassingsgebied van [begrip] is duidelijk afgebakend."

Dependencies

Notes
