US-190: Orchestration Tab (V2) – UI Koppeling met iteraties

Doel

Koppel de bestaande orchestration tab aan de V2‑orchestrator iteratieloop (US‑188) en FeedbackBuilder V2 (US‑189), toon iteratie‑metrics (scores, violations per iteratie, duur) en diff t.o.v. vorige iteratie. Achter feature‑flag.

Acceptatiecriteria

  • [ ] UI gebruikt uitsluitend V2‑API’s en V2‑UIResponse; geen legacy imports.
  • [ ] Visualisaties: score per iteratie (lijn/balk), violations per severity, diff van definitietekst.
  • [ ] Telemetrie zichtbaar (duur, feedbackcount, reused_examples indicator).
  • [ ] Feature‑flag gated; verborgen in standaardconfig.

Definition of Done

  • [ ] UI‑componenten en wiring naar services (async via UI‑bridge waar nodig).
  • [ ] Flag‑toggle en duidelijke “Demo (V2 Iterations)” label.
  • [ ] Tests: smoke‑flow met mocks, rendering‑sanity.

Technical Notes

  • Gebruik `ui/helpers/async_bridge.py` voor async‑aanroepen.
  • Geen sessiestate voor bron van waarheid; alles via services.
  • Exporteer samenvatting naar JSON (optioneel) voor debug.

Testplan

  • Smoke: 1–3 iteratie scenario (mocks) met vaste resultaten; visualisaties tonen correcte waarden.
  • Contract: UI leest alleen V2‑keys en faalt niet bij ontbrekende legacy velden.

Risico’s

  • Verwarring met legacy tab → duidelijk label en flag; documenteer verschil.
