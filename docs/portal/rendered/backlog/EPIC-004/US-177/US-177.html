US-177: UI Container introduceren en ServiceContainer ontkoppelen van UI

Doel: UI specifieke services in src/ui/container.py; src/services/container.py beheert alleen services.

Scope

  • Nieuwe `src/ui/container.py` met creatie van `DefinitionUIService` e.d.
  • Verwijder UI-service creatie uit `src/services/container.py`.
  • Pas UI imports aan naar `ui.container` i.p.v. services-container.

Acceptatiecriteria

  • `rg -n "\bfrom\s+ui\.|\bimport\s+ui\b" src/services` → 0 matches
  • UI blijft functioneren met nieuwe container.

Taken

1) Introduceer UI-container en verplaats constructie van DefinitionUIService.

2) Update UI callsites; smoke-test.

Testplan

  • Unit: import van services-container faalt niet door UI.
  • Smoke: UI-tabs laden en kunnen exporteren via UI-service.
