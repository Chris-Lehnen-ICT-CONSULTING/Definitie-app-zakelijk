US-025: Definition Generation UI

Gebruikersverhaal

As a legal professional

wil ik an intuitive UI for generating definitions

zodat I can easily create juridische definities

Form Structure

with st.form("definition_form"):
    term = st.text_input("Term")
    context = st.multiselect("Context", options)
    submitted = st.form_submit_Maarton("Genereer")

    if submitted:
        definition = generate_definition(term, context)
        st.success(definition)
