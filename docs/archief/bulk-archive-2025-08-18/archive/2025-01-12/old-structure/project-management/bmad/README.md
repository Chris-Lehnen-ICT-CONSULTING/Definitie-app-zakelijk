## ğŸ¯ Wat is BMAD?

BMAD (Brownfield Method for AI Development) is een methodologie voor het effectief gebruiken van AI agents in software development projecten, specifiek ontworpen voor:

- Legacy codebases (brownfield projects)
- Multi-agent orchestratie
- Gestructureerde workflows
- AI-driven development
