# US-099: Traceability‑integratie in generator (REQ↔EPIC↔US)

## Gebruikersverhaal
**Als** architect wil ik dat de generator `docs/traceability.json` (en REQ‑REGISTRY) inleest, **zodat** de Portal REQs kan tonen met gekoppelde EPICs/US en omgekeerd.

## Acceptatiecriteria
- [ ] Generator leest `docs/traceability.json` (indien aanwezig) en voegt relaties toe aan items
- [ ] Voor REQ: lijst EPICs/US (ids) zichtbaar in JSON en UI (links)
- [ ] Voor EPIC: lijst REQs/US zichtbaar
- [ ] Fallback: als traceability.json ontbreekt, degradeert UI zonder fouten

## Implementatienotities
- Minimalistische parser; respecteer bestaande schema
- Verrijk `documents[i].relations = { reqs, epics, stories }`

## Verificatie
- Voorbeeld met een paar REQs/EPICs/US toont juiste koppelingen en klikbare links
