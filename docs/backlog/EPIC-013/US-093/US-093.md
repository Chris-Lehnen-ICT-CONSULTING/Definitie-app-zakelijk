# US-093: Herbruikbare packaging & configuratie voor andere projecten

## Gebruikersverhaal
**Als** platformteam wil ik de portalgenerator als herbruikbare oplossing aanbieden, **zodat** andere projecten met minimale setup dezelfde portal kunnen gebruiken.

## Acceptatiecriteria
- [ ] Generator is te gebruiken via `pipx`/`pip` of als vendored script met config (`docs/portal/config.yaml`)
- [ ] Repostructuur‑agnostisch: projectpaden en categorieën via config
- [ ] Voorbeeldimplementatie + handleiding voor integratie in een tweede project

## Implementatie‑notities
- Publiceer als interne Python package of subrepo/template
- Config‑sleutels: include/exclude globs, sectiemapping, badges, paden

## Verificatie
- Test in sandboxrepo: portal werkt met alleen config aanpassing
