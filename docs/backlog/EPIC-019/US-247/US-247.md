# US-019-10 — Templates versimpelen

Als schrijver wil ik per categorie één kern‑template en maximaal twee patronen, zodat aanwijzingen duidelijk en kort zijn.

## As‑Is
- TemplateModule toont template + meerdere patronen + voorbeelden per categorie.

Bronnen:
- src/services/prompts/modules/template_module.py:87, 145, 170, 209

## To‑Be
- Eén kern‑template + max 2 patronen; geen voorbeelden (standaard).

## Acceptance Criteria
- AC1: Templatesectie ≤ ~10 regels.
- AC2: Geen “Voorbeelden…” sectie in prompt (standaard).

## Implementatie (richting)
1) template_module: `include_examples=False` default; slice patronen tot 2.
2) Config via ModularPromptAdapter mapping.

## Testplan
- Snapshot: sectie kort; geen voorbeelden; 2 patronen max.

## Risks / Mitigations
- Risico: te weinig guidance bij sommige categorieën
  - Mitigatie: toggle via config; UI‑help kan uitgebreide voorbeelden tonen.
