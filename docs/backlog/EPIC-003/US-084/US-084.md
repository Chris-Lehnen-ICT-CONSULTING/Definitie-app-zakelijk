# US-084: DBpedia Integratie (SPARQL)

## Gebruikersverhaal
**Als** ontwikkelaar  
**wil ik** standaardkenmerken van begrippen uit DBpedia kunnen ophalen  
**zodat** definities aanvullende context (type/categorie/relaties) kunnen tonen

## Acceptatiecriteria
- [ ] SPARQL‑query voor basiskenmerken (type/categorie/related)  
- [ ] Fallback‑provider (lage prioriteit) wanneer Wikipedia geen structuur biedt  
- [ ] Provenance (resource‑URI, URL) wordt vastgelegd en getoond

### Niet‑functionele acceptatiecriteria (Veilige Scraping)
- Respecteer rate limits van endpoint; timeout ≤ 5s; exponential backoff  
- Sanitization van responses; geen PII; AVG/BIR‑conform  
- Eenduidige User‑Agent; volledige provenance; beknopte audit logging
