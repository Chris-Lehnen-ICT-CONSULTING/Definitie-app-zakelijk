## Business Context
Set van 6 structure checks die de grammaticale en structurele correctheid van definities valideren.

## Acceptance Criteria
- [ ] Check max_one_sentence (enkele zin)
- [ ] Check clear_structure (min 5 woorden)
- [ ] Check clear_pronoun_references
- [ ] Check clear_article_references
- [ ] Check proper_noun_start
- [ ] Check concrete_terminology

## Structure Checks
```python
checks = {
    "max_one_sentence": lambda d: len(re.split(r"[.!?]+", d.strip())) <= 2,
    "clear_structure": lambda d: len(d.split()) >= 5,
    "clear_pronoun_references": lambda d: not re.search(
        r"\b(deze|dit|die)\b(?!\s+(begrip|definitie))", d
    ),
    "clear_article_references": lambda d: not re.search(
        r"\bde\s+(proces|activiteit)\b(?!\s+\w+)", d
    ),
    "proper_noun_start": lambda d: not re.match(
        r"^(is|de|het|een|wordt|betreft|zijn|kan|moet|mag)\b", d.strip()
    ),
    "concrete_terminology": lambda d: not re.search(
        r"\b(proces|activiteit|zaak|ding)\b(?!\s+\w+)", d
    )
}
```
