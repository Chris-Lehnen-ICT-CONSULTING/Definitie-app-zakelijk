# US-116: Implementeer Linguistic Enhancer

## User Story
**Als** taal optimizer  
**Wil ik** taalkundige kwaliteit van definities verbeteren  
**Zodat** ze formeler en consistenter zijn  

## Status
**Status**: TODO  
**Priority**: LOW

## ⚠️ WAARSCHUWING
Mogelijk overbodig door moderne AI capabilities. Evalueer necessity.

## Business Context
Verbetert taalkundige aspecten zoals actieve vorm, formaliteit, en redundantie verwijdering.

## Acceptance Criteria
- [ ] Passive naar active voice conversie
- [ ] Redundantie detectie en verwijdering
- [ ] Formalisering (je/jij → men)
- [ ] Begrip simplificatie ("het begrip X" → "X")
- [ ] Confidence score voor elke wijziging

## Linguistic Patterns
```python
improvements = {
    "passive_to_active": [
        (r"\bwordt\s+(\w+)", r"(\w+) wordt")
    ],
    "redundancy": [
        (r"\b(\w+)\s+\w*\s+\1\b", "Word repetition"),
        (r"\bhet\s+begrip\s+(\w+)", r"\1")
    ],
    "formality": [
        (r"\bje\b", "men"),
        (r"\bjij\b", "men"),
        (r"\bwij\b", "men")
    ]
}
```

## Implementation
```python
def enhance_linguistics(text: str) -> tuple[str, list[Change]]:
    enhanced = text
    changes = []
    
    for category, patterns in improvements.items():
        for pattern, replacement in patterns:
            new_text = re.sub(pattern, replacement, enhanced)
            if new_text != enhanced:
                changes.append(Change(category, confidence=0.7))
                enhanced = new_text
    
    return enhanced, changes
```
