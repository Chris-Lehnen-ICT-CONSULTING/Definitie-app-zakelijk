## Business Context
Detecteert en verbetert vage terminologie in definities. Gebruikt pattern matching en vagueness scoring.

## Acceptance Criteria
- [ ] Detecteer vague patterns (enigszins, wellicht, etc.)
- [ ] Bereken vagueness score (0.0-1.0)
- [ ] Trigger enhancement bij score > 0.3
- [ ] Detecteer en fix cirkelredenering
- [ ] Vervang vage termen met specifieke alternatieven

## Vagueness Patterns
```python
unclear_patterns = [
    r"\b(enigszins|soms|wellicht|mogelijk|misschien)\b",
    r"\b(enzovoort|etc\.?|enz\.?)\b",
    r"\b(diverse|verschillende|meerdere)\b(?!\s+\w+)",
    r"\b(dergelijke|soortgelijke)\b(?!\s+\w+)"
]
```

## Scoring Algorithm
```python
vagueness_score = matches / max(len(text.split()), 1) * 10
# Cap at 1.0
```

## Circular Reasoning Detection
- Check: begrip komt >1 keer voor
- Fix: vervang 2e+ occurrence met "dit"
