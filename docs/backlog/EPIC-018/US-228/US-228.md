# US-228: UI‑feedback dat documentcontext is gebruikt

## Gebruikersverhaal
Als gebruiker wil ik in de UI terugzien dat mijn documentcontext is gebruikt, zodat ik vertrouwen heb in de uitkomst.

## Acceptatiecriteria
- [ ] Succesmelding toont: “met context van X document(en)”
- [ ] Resultaatweergave bevat indicator/badge dat documentcontext is meegenomen
- [ ] Logging bevestigt gebruik (zonder PII)

## Tests
- UI smoke: indicator aanwezig bij geselecteerde documenten
