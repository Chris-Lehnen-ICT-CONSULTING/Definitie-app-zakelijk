# US-229: Optioneel — document‑snippets in promptinjectie

## Gebruikersverhaal
Als gebruiker wil ik korte fragmenten uit mijn documenten terugzien als “Bron” in de prompt, zodat context nog explicieter wordt gebruikt.

## Acceptatiecriteria
- [ ] Documenten kunnen als 1–2 “Bron”‑regels worden geïnjecteerd (sanitize + truncation)
- [ ] Token‑budget en max_snippets gerespecteerd
- [ ] Config‑toggle om feature aan/uit te zetten

## Technisch
- Zelfde schema als web_lookup sources (provider/title/url/snippet/score/used_in_prompt)
- Provider = “documents”

## Tests
- Prompt bevat maximaal N document‑snippets binnen tokenbudget
