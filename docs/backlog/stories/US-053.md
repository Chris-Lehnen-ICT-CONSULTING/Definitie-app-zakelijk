### WCAG Categorieën

#### 1. Perceivable (Waarneembaar)
**Definitie Generator Specifiek:**
- **1.1 Text Alternatives**
  - Alt text voor validatie iconen
  - Beschrijvingen voor score visualisaties
  - Tekstuele weergave van progress indicators

- **1.3 Adaptable**
  - Semantische HTML voor definitie structuur
  - Proper heading hierarchy (h1-h6)
  - Meaningful sequence voor validatie resultaten

- **1.4 Distinguishable**
  - Contrast ratio 4.5:1 voor normale tekst
  - Contrast ratio 3:1 voor grote tekst
  - Niet alleen kleur voor validatie status
  - Resizable text tot 200% zonder horizontaal scrollen

#### 2. Operable (Bedienbaar)
**Juridische Applicatie Focus:**
- **2.1 Keyboard Accessible**
  - Alle interactieve elementen keyboard bereikbaar
  - Tab order logisch voor definitie workflow
  - Keyboard shortcuts voor frequent acties
  - Geen keyboard traps in modals/expanders

- **2.2 Enough Time**
  - Configureerbare session timeouts
  - Waarschuwing voor timeout met verlenging optie
  - Geen tijdslimiet voor definitie generatie

- **2.4 Navigable**
  - Skip links naar hoofdcontent
  - Breadcrumbs voor context
  - Focus indicators duidelijk zichtbaar
  - Page titles beschrijvend en uniek

#### 3. Understandable (Begrijpelijk)
**Nederlandse Juridische Context:**
- **3.1 Readable**
  - Taal declaratie (lang="nl")
  - Juridische termen uitgelegd
  - B1 taalniveau waar mogelijk
  - Glossary voor vaktermen

- **3.2 Predictable**
  - Consistente navigatie
  - Consistente terminologie
  - Geen onverwachte context changes

- **3.3 Input Assistance**
  - Duidelijke error messages
  - Input format hints
  - Inline validatie feedback
  - Success confirmations

#### 4. Robust (Robuust)
**Technische Implementatie:**
- **4.1 Compatible**
  - Valid HTML waar Streamlit toelaat
  - Proper ARIA gebruik
  - Werkt met NVDA, JAWS, VoiceOver
  - Progressive enhancement strategie
