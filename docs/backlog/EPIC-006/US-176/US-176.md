# US-176: SecurityMiddleware minimale integratie aan UI-zijde

Doel: basis security preflight op UI-entrypoints naar orchestrators zonder services te vervuilen.

## Scope
- `src/ui/**` entrypoints die orchestrator-calls doen: roep `get_security_middleware().validate_request(...)` aan met `ValidationRequest`.
- Logging/headers voor audit; geen service-injectie in services.

## Acceptatiecriteria
- Zichtbare usage in UI main pad; geen usages in `src/services/**`.
- Unit/integratietest: malafide input geblokkeerd; normale input toegestaan binnen latency-budget.

## Taken
1) Voeg preflight call(s) toe rond generate/validate/export acties.
2) Documenteer configuratie (env vars/flags) in bestaande security-epic docs.
