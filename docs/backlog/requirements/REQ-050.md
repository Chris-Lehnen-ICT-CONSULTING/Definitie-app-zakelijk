# Requirement REQ-050: Accessibility WCAG 2.1 AA compliance

## Beschrijving
Het systeem moet voldoen aan WCAG 2.1 niveau AA richtlijnen voor web accessibility OM toegankelijk te zijn voor gebruikers met beperkingen.

## Rationale
Wettelijk verplicht voor overheidssystemen onder de Wet digitale toegankelijkheid overheid. Zorgt voor inclusieve toegang tot de applicatie.

## Acceptatiecriteria (SMART)
- Keyboard navigatie voor alle functies
- Screen reader compatibility (ARIA labels)
- Color contrast ratio minimaal 4.5:1
- Focus indicators zichtbaar
- Alt text voor alle images
- Accessibility audit score > 90%
