# Requirement REQ-034: Custom Rule Configuration

## Beschrijving
Het systeem moet gebruikers in staat stellen OM validatieregels te configureren, inclusief het aan/uitzetten van specifieke regels en het aanpassen van regel-parameters.

## Rationale
Verschillende juridische domeinen kunnen verschillende validatie-eisen hebben. Configureerbare regels maken het systeem flexibel voor verschillende use cases binnen de overheid.

## Acceptatiecriteria (SMART)
- UI voor regel configuratie beschikbaar
- Regels kunnen individueel aan/uit gezet worden
- Parameters per regel aanpasbaar (bijv. max lengte)
- Configuraties persistent opgeslagen
- Import/export van regel configuraties mogelijk

- **Relevant:** Direct gerelateerd aan gebruikersbehoefte
- **Tijdgebonden:** Gerealiseerd binnen huidige sprint


## Uitwerking / Notities
Configuratie opties:
- Enable/disable per regel
- Severity level override
- Custom thresholds (bijv. max woorden)
- Domain-specifieke regel sets
