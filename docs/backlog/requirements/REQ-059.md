# Requirement REQ-059: Environment-based Configuration

## Beschrijving
Het systeem moet verschillende configuraties ondersteunen voor development, testing, acceptance, en production environments met veilige secret management.

## Rationale
Environment-specific configuratie voorkomt beveiliging issues (productie credentials in development) en maakt gecontroleerde deployments mogelijk.

## Acceptatiecriteria (SMART)
- Separate config files per environment
- Environment variables voor secrets
- Configuration validatie bij startup
- Override mogelijkheid voor testing
- No hardcoded values in code

1. Default configuration
2. Environment-specific config
3. Environment variables
4. CommEn-line arguments

Secrets via environment variables only.
