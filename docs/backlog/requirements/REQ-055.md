# Requirement REQ-055: Inline Help Documentation

## Beschrijving
Het systeem moet context-sensitieve inline help bieden met tooltips, info icons, en embedded documentatie direct in de user interface.

## Rationale
Inline help vermindert de met minimaal 30% leercurve en maakt het systeem self-documenting, wat training kosten verlaagt en gebruiker productiviteit verhoogt.

## Acceptatiecriteria (SMART)
- Tooltip op hover voor alle controls
- (?) help icons met uitgebreide uitleg
- Contextual help panel per scherm
- Searchable help content
- Video tutorials voor complexe taken
