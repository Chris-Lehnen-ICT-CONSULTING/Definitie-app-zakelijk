# Requirement REQ-103: Externe Bronnen Beheer UI

## Beschrijving
UI‑componenten voor bronbeheer: tonen van geregistreerde bronnen (id, type, status), verbinding testen, verwijderen/deregistreren en toevoegen van nieuwe bronnen (mock en bestandssysteem upload).

## Acceptatiecriteria (SMART)
- Specifiek: lijstweergave + acties (Test, Verwijder), knoppen voor Mock en File‑bron toevoegen
- Meetbaar: statusindicator werkt (verbonden/niet), fouten zichtbaar in UI
- Acceptabel: wijzigingen direct zichtbaar; geen page reload errors
- Relevant: minimaliseert drempel om externe bronnen te gebruiken
- Tijdgebonden: MVP bij EPIC‑022
