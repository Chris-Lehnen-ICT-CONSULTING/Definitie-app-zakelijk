# US-125: Role-Based Access Control (RBAC)

Niet nodig voor single‑user; gepland voor multi‑user. Rollen/permissions en protectie van gevoelige functies.

## Acceptatiecriteria (functioneel)
- [ ] Rollen: Admin/Editor/Viewer; permissies gedefinieerd per rol
- [ ] Beschermd pad voor admin‑functies (config, API‑keys, workflows)
- [ ] RBAC decision gelogd (resourcenaam/uitkomst)

## Niet‑functioneel
- [ ] Least‑privilege default; deny‑by‑default policy
- [ ] Testmatrix voor kernpaden (toegang/geen toegang)

## Definition of Done
- [ ] Documentatie van rollen en permissies
- [ ] Unit/integratie tests voor 3 kernpaden

## Afhankelijkheden
- US‑124 (auth), US‑126 (API‑keys)
