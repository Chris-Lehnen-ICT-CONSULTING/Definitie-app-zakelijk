# US-167: Autorisatie voor Beheer (beheerderrol)

## Doel
Toegang tot Beheer‑tab en configuratiewijzigingen beperken tot geautoriseerde beheerders; audit alle beheeracties.

## Acceptatiecriteria
- [ ] Beheer‑tab alleen zichtbaar voor beheerderrol (feature‑flag tijdelijk toegestaan)
- [ ] Opslaan van wijzigingen alleen toegestaan voor beheerderrol
- [ ] Audit van alle beheeracties met user/tijd/actie/details
- [ ] Voor nu: simpele rol via env/flag; later integratie met echte auth

## Definition of Done
- [ ] Feature‑flag/rol‑check geïmplementeerd
- [ ] UI‑knoppen disabled/hidden voor niet‑beheerders
- [ ] Audit aanwezig
- [ ] Documentatie bijgewerkt
