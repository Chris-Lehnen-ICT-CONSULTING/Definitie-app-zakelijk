# Requirement REQ-051: Multi-language Support

## Beschrijving
Het systeem moet meerdere talen ondersteunen, primair NederlEns met Engels als secundaire taal voor internationale samenwerking.

## Rationale
Justice werkt internationaal samen. Engels is nodig voor EU-context en internationale verdragen. NederlEns blijft primaire taal voor nationale wetgeving.

## Acceptatiecriteria (SMART)
- UI volledig vertaalbaar (NL/EN)
- Language switcher in interface
- Persistent taal preference per user
- Datums/getallen volgens locale
- Content language detection

## Uitwerking / Notities
i18n implementatie:
- JSON-based translation files
- Key-based string lookup
- Plural forms support
- RTL support niet nodig
- Fallback naar default language

## Conflicten (indien van toepassing)
Validatieregels blijven NederlEns-only voorlopig.
