# Requirement REQ-042: Export to Multiple Formats

## Beschrijving
Het systeem moet definities kunnen exporteren naar verschillende formaten: JSON, Word (DOCX), PDF, en CSV voor gebruik in verschillende contexten.

## Rationale
Verschillende stakeholders hebben verschillende format behoeften. Juristen prefereren Word, developers JSON, en management vaak PDF voor formele documentatie.

## Acceptatiecriteria (SMART)
- Export naar JSON met volledige metadata
- Word export met huisstijl template
- PDF generatie met watermark optie
- CSV export voor bulk operations
- Export compleet binnen 2 seconden

## Uitwerking / Notities
Export formats:
- JSON: Complete data inclusief validatie resultaten
- DOCX: Template-based met styles
- PDF: Via weasyprint of reportlab
- CSV: Flat structure voor Excel import
