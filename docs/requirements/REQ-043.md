# Requirement REQ-043: Import from External Sources

## Beschrijving
Het systeem moet definities kunnen importeren uit CSV en JSON bestanden, met validatie en mapping van externe data naar het interne datamodel.

## Rationale
Import functionaliteit maakt bulk migratie mogelijk van bestaande definities en integratie met externe systemen die definities aanleveren.

## Acceptatiecriteria (SMART)
- CSV import met header mapping
- JSON import met schema validatie
- Duplicate detection bij import
- Validation van geïmporteerde definities
- Import rapport met succes/failure status

## Uitwerking / Notities
Import features:
- Flexible field mapping UI
- Data cleaning en normalisatie
- Batch import tot 1000 items
- Rollback bij failures
- Import history tracking
