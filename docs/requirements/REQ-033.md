# Requirement REQ-033: Rule Conflict Resolution

## Beschrijving
Het systeem moet conflicten tussen validatieregels detecteren en oplossen wanneer verschillende regels tegenstrijdige feedback geven over dezelfde definitie-aspecten.

## Rationale
Validatieregels kunnen soms conflicteren (bijv. STR-01 vereist kortheid, ESS-02 vereist volledigheid). Een conflict resolution mechanisme voorkomt verwarrende feedback aan gebruikers.

## Acceptatiecriteria (SMART)
- Conflict detectie identificeert overlappende regel-feedback
- Priority-based resolution (high priority wint)
- Conflict rapport toont welke regels conflicteren
- Gebruiker kan conflicts manueel overrulen
- Conflict resolution binnen 50ms

## Uitwerking / Notities
Conflict resolution strategie:
1. Detecteer overlappende text ranges
2. Vergelijk regel priorities
3. Behoud hoogste priority feedback
4. Log conflicts voor analyse

## Conflicten (indien van toepassing)
Samenwerking nodig met REQ-030 (priority system) voor consistente priority handling.
