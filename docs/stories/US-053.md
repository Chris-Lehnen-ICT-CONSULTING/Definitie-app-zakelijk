**Wanneer** ik de richtlijnen raadpleeg

### Wettelijke Vereisten

- EN 301 549 standaard

### WCAG Categorieën
- **Perceivable**: Alternatieve content, contrast ratios
- **Operable**: Keyboard navigatie, timing, seizures
- **Understandable**: Taal, voorspelbaarheid, input hulp
- **Robust**: Parsing, compatibility met assistive tech

### Prioriteit Items
1. Keyboard navigatie (Tab order, shortcuts)
2. Screen reader compatibility (ARIA labels)
3. Kleurcontrast (4.5:1 voor normale tekst)
4. Focus indicators (visible, high contrast)
5. Error messages (descriptive, linked to fields)
6. Form labels (associated, descriptive)
7. Skip links (to main content)
8. Language declarations (lang="nl")

## Test Scenario's
1. Volledige keyboard-only navigatie mogelijk
2. Screen reader kan alle content voorlezen
3. Kleurcontrast voldoet aan 4.5:1 ratio
4. Geen keyboard traps
5. Time-outs kunnen verlengd worden
6. Formulieren hebben proper labels
7. Error messages zijn duidelijk gekoppeld
